# Instrucciones para ejecutar el Frontend

Este proyecto es una aplicación React construida con Vite. Sigue estos pasos para ejecutarla:

## Requisitos previos

Necesitas tener **Node.js** instalado en tu computadora.
Si no lo tienes, descárgalo e instálalo desde: https://nodejs.org/

## Pasos para ejecutar

1.  Abre la carpeta `product-tracker` dentro de esta carpeta `FRONTEND`.
2.  Haz clic derecho en un espacio vacío dentro de la carpeta y selecciona "Abrir en Terminal" (o abre PowerShell y navega hasta aquí).
3.  Instala las dependencias ejecutando el siguiente comando:
    ```bash
    npm install
    ```
4.  Una vez instaladas, inicia el servidor de desarrollo con:
    ```bash
    npm run dev
    ```
5.  Mantén la terminal abierta. Verás una dirección local (por ejemplo, `http://localhost:5173`).
6.  Abre tu navegador web y escribe esa dirección para ver la aplicación.

¡Listo!
